# 离火区 聚落档案

## 基本资料

**名称**: 阳爻锻炉镇  
**聚落类型**: 工业城镇  
**面积**: 8km²  
**聚落负责人**: 卦中人·祝融  

## 聚落简介

岩浆湖旁的锻造中心，由卦中人操控机械运作，阴阳聚变炉和微波输电装置所在区域。
