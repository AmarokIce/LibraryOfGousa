# 坎水区 区域档案

## 基本资料

**名称**: 坎水区（地下）  
**面积**: 约21万平方公里  
**探索度**: 13%  
**危险系数**: 75%  
**地区负责人**: 卦中人·禹，卦中人·共工

## 区域简介

原为天秦生态保护区，现被逆模因污染区，会吞噬记忆的黑色水域，任何光学记录设备在此都会输出混乱卦象。影商人·晦在此出没，为来访者提供一些物品，至于回报——他并不需要什么，却依旧喜欢让你感觉自己损失了部分东西。

*“可不能白嫖!”*
